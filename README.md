
# Installation et exécution de ce projet Blazor *nextdevblazor-app*

## 1. Prérequis techniques
Avant toute chose, vérifier que l’environnement est conforme.
### Outils requis
- **.NET SDK 9** 
- **Git**
- **IDE recommandé** : Visual Studio 2022+ ou VS Code

### Vérification
```bash
dotnet --version
git --version
```

## 2. Récupération du projet
Cloner le dépôt GitHub :
```bash
git clone https://github.com/akaletekoffilevis/nexdevblazor-app.git
cd nextdevblazor-app
```

## 3.Restauration des dépendances
Exécuter à la racine du projet :
```bash
dotnet restore
```



## 4.Lancer l’application
```bash ou cmd
dotnet run
```
